API 使用文档

所有 API 通过 X-Api-Key Header 认证

认证方式

在所有请求中添加 Header：

X-Api-Key: djd_YOUR_API_KEY_HERE

📦 存储桶 API

{% with base_url=base_url|add:'' %}
GET {{ base_url }}/buckets/api/buckets/ 列出所有桶
返回用户的所有存储桶列表（JSON）。
POST {{ base_url }}/buckets/api/buckets/create/ 创建桶
Body (JSON): {"name": "my-bucket", "is_public": false}
DELETE {{ base_url }}/buckets/api/buckets/<id>/ 删除桶
删除整个桶及其所有文件（不可恢复）。
GET {{ base_url }}/buckets/api/buckets/<id>/files/ 列出桶内文件
可选 ?path=dir1/ 浏览子目录。
POST {{ base_url }}/buckets/api/buckets/<id>/upload/ 上传文件
multipart/form-data: files (多文件), folder_path (可选)
DELETE {{ base_url }}/buckets/api/buckets/<id>/files/<fid>/ 删除文件
POST {{ base_url }}/buckets/api/buckets/<id>/folder/create/ 创建目录
Body (JSON): {"name": "photos", "parent_path": ""}
DELETE {{ base_url }}/buckets/api/buckets/<id>/folder/?path=dir1/ 删除目录
{% endwith %}

📂 文件管理 API

操作"我的文件"中的文件和文件夹。

{% with base_url=base_url|add:'' %}
GET {{ base_url }}/files/api/list/
列出文件和文件夹。可选 ?folder=<id>
POST {{ base_url }}/files/api/upload/
multipart/form-data: files, folder (可选)
GET {{ base_url }}/files/api/<id>/download/
下载文件（FileResponse）。
DELETE {{ base_url }}/files/api/<id>/delete/
软删除文件（移至回收站）。
POST {{ base_url }}/files/api/folder/create/
Body (JSON): {"name": "docs", "parent": "<id>"}
{% endwith %}

curl 快速测试

# 列出桶
curl -H "X-Api-Key: djd_YOUR_KEY" {{ base_url }}/buckets/api/buckets/

# 创建桶
curl -X POST -H "X-Api-Key: djd_YOUR_KEY" -H "Content-Type: application/json" \
  -d '{"name":"test","is_public":false}' {{ base_url }}/buckets/api/buckets/create/

# 上传文件
curl -X POST -H "X-Api-Key: djd_YOUR_KEY" \
  -F "files=@hello.txt" -F "folder_path=" {{ base_url }}/buckets/api/buckets/<id>/upload/

# 下载文件
curl -H "X-Api-Key: djd_YOUR_KEY" -o output.txt \
  {{ base_url }}/files/api/<id>/download/
